# Daily Agent Workflow

This workflow outlines daily responsibilities for an agent working on the NovaCore project.

1. **Morning Review:** Examine open issues, recent pull requests, and project status updates. Plan tasks for the day.
2. **Development Block:** Implement or refine a single feature aligned with current milestones (e.g., mesh rendering, character controller). Follow the recommended workflow in AGENTS.md and the other lists: read code, make focused changes, build, run tests.
3. **Testing & Validation:** Run smoke tests and any additional unit tests for the affected subsystem. Analyze results and fix regressions.
4. **Documentation & Communication:** Update relevant documentation (architecture docs, status files) if changes impact design or progress. Document progress and next steps.
5. **End-of-Day Sync:** Commit and push code with a clear message. Summarize completed work, open questions, and tasks for the next day.

Adjust the schedule based on the project’s priorities and milestones.
